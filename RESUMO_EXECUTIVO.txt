================================================================================
                    CONCEITOS-CHAVE
================================================================================

🎯 **ERRO TRANSIENTE vs ERRO PERMANENTE:**
   
   TRANSIENTE (ProcessingException) → RETRY
   ├─ Servidor não está a correr
   ├─ Timeout (connect ou read)
   ├─ Conexão TCP perdida
   └─ Anomalia de rede
   
   PERMANENTE (Resposta HTTP 4xx/5xx) → NÃO RETRY
   ├─ 400 BAD_REQUEST (parâmetros inválidos)
   ├─ 403 FORBIDDEN (autenticação falhou)
   ├─ 404 NOT_FOUND (recurso não existe)
   └─ 409 CONFLICT (já existe)

🗃️ **HIBERNATE ORM:**
   ├─ Mapeia objetos Java → Tabelas SQL
   ├─ Anotações: @Entity, @Id, @Column, @ElementCollection
   ├─ JPQL: Query language independente da BD
   └─ Gestão automática de transações

┌─────────────────────────────────────────────────────────────────────────┐
│  TENTATIVA 1                                                            │
│  ├─ Envia request REST                                                  │
│  ├─ Recebe resposta? → SIM → Retorna resultado                         │
│  └─ ProcessingException? → SIM → Continua                              │
│                                                                          │
│  ⏰ SLEEP 5 segundos                                                     │
│                                                                          │
│  TENTATIVA 2                                                            │
│  ├─ Envia request REST                                                  │
│  ├─ Recebe resposta? → SIM → Retorna resultado                         │
│  └─ ProcessingException? → SIM → Continua                              │
│                                                                          │
│  ⏰ SLEEP 5 segundos                                                     │
│                                                                          │
│  TENTATIVA 3                                                            │
│  ├─ Envia request REST                                                  │
│  ├─ Recebe resposta? → SIM → Retorna resultado                         │
│  └─ ProcessingException? → SIM → Continua                              │
│                                                                          │
│  ❌ Esgotou tentativas → Retorna Result.error(TIMEOUT)                  │
└─────────────────────────────────────────────────────────────────────────┘

Tempo máximo: 3 × (5s connect + 5s read) + 2 × 5s sleep = 40 segundos
